```markdown
# macOS Terminal Commands Cheat Sheet

## Navigation
- `cd [directory]`: Change directory.
- `cd ~`: Go to the home directory.
- `cd ..`: Go up one directory level.
- `ls`: List files and directories.
- `ls -l`: Detailed list view.
- `pwd`: Print working directory.

## File Management
- `touch [filename]`: Create a new file.
- `mkdir [directory]`: Create a new directory.
- `cp [source] [destination]`: Copy file/directory.
- `mv [source] [destination]`: Move file/directory.
- `rm [file]`: Remove file (use with caution).
- `rm -r [directory]`: Remove directory and its contents (use with extreme caution).

## Viewing and Editing Files
- `cat [filename]`: Display file contents.
- `less [filename]`: View file contents with paging.
- `nano [filename]`: Open file for editing in Nano editor.
- `vim [filename]`: Open file for editing in Vim editor.

## System Information
- `top`: Display real-time system usage.
- `df -h`: Show disk space usage.
- `du -h [directory]`: Show directory space usage.
- `uname -a`: Display system information.

## Process Management
- `ps`: List active processes.
- `kill [PID]`: Terminate process by ID.
- `killall [process name]`: Terminate all processes with name.
- `bg`: Background a process.
- `fg`: Bring a background process to the foreground.

## Networking
- `ping [host]`: Send ICMP echo request to host.
- `ifconfig`: Show network interfaces.
- `netstat`: Display network connections, routing tables, interface statistics, masquerade connections, and multicast memberships.

## Package Management
- `brew install [package]`: Install package with Homebrew.
- `brew update`: Update Homebrew and packages.
- `brew upgrade [package]`: Upgrade installed package.
- `brew list`: List installed packages.

## Miscellaneous
- `sudo [command]`: Run command with superuser privileges.
- `history`: Show command history.
- `clear`: Clear the terminal screen.
```
